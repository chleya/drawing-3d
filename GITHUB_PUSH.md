# Drawing 3D - GitHub Push Script
# 推送到GitHub

## 1. 创建GitHub仓库

```bash
# 创建仓库
gh repo create drawing-3d --public --source=. --description "Road Engineering Management System"

# 推送
git add .
git commit -m "Initial commit: Drawing 3D v1.0"
git push -u origin main
```

## 2. 或者使用Git手动

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/chleya/drawing-3d.git
git push -u origin main
```
